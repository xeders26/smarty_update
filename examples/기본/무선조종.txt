<h2 style="color: #3498db; margin: 0 0 5px 0;">📱 고급 통신: 연속 명령어(패킷) 처리하기</h2>
<hr style="margin: 5px 0 15px 0; border-color: #555;">

<p style="margin: 0 0 15px 0; line-height: 1.5;">스마트폰에서 블루투스로 <b>두 개의 글자를 연속으로 전송(예: "Fa", "Fs")</b>했을 때, 이를 순서대로 읽어서 조합에 따라 로봇의 LED를 제어하는 고급 예제입니다.</p>

<h3 style="color: #3498db; margin: 0 0 10px 0;">🧩 블록 동작 흐름</h3>
<ol style="line-height: 1.5; margin: 0 0 15px 25px; padding: 0;">
  <li style="margin-bottom: 5px;"><b>첫 번째 글자 읽기:</b> 블루투스 데이터가 들어오면 <b>첫 번째 글자</b>를 읽어서 <code style="color: #f1c40f;">명령</code> 변수에 저장합니다.</li>
  <li style="margin-bottom: 5px;"><b>두 번째 글자 읽기:</b> 그 뒤에 또 데이터가 들어왔는지 확인하고, <b>두 번째 글자</b>를 읽어서 <code style="color: #f1c40f;">값</code> 변수에 저장합니다.</li>
  <li style="margin-bottom: 5px;"><b>명령어 조합 분석 (만약 ~라면):</b>
    <ul style="margin: 5px 0 5px 20px; padding: 0;">
      <li style="margin-bottom: 3px;"><b>"Fa" 가 들어왔을 때:</b> (<code style="color: #f1c40f;">명령</code>이 'F' <b>그리고</b> <code style="color: #f1c40f;">값</code>이 'a' 라면) 👉 <b>내장 LED1을 켜고</b>, LED2는 끕니다. 범퍼 LED도 끕니다.</li>
      <li style="margin-bottom: 3px;"><b>"Fs" 가 들어왔을 때:</b> (<code style="color: #f1c40f;">명령</code>이 'F' <b>그리고</b> <code style="color: #f1c40f;">값</code>이 's' 라면) 👉 내장 LED1을 끄고, <b>내장 LED2를 켭니다.</b> 범퍼 LED도 끕니다.</li>
      <li style="margin-bottom: 3px;"><b>그 외의 다른 글자일 때:</b> (예: "Fq", "Ab" 등) 👉 내장 LED 대신 <b>범퍼 LED에 불을 켭니다.</b></li>
    </ul>
  </li>
</ol>
