<h2 style="color: #3498db; margin: 0 0 5px 0;">💡 기초 예제: 내장 LED 깜빡이기</h2>
<hr style="margin: 5px 0 15px 0; border-color: #555;">

<p style="margin: 0 0 15px 0; line-height: 1.4;">스마티 로봇과 처음 인사하는 가장 기초적인 예제입니다. 로봇의 심장이 뛰듯 <b>LED가 1초마다 켜지고 꺼지기를 무한히 반복</b>합니다.</p>

<h3 style="color: #3498db; margin: 0 0 10px 0;">🧩 블록 동작 흐름</h3>
<ol style="line-height: 1.4; margin: 0 0 15px 25px; padding: 0;">
  <li style="margin-bottom: 5px;"><b>스마티 시작하기 (setup):</b> 전원이 켜지면 로봇을 초기화합니다.</li>
  <li style="margin-bottom: 5px;"><b>내장 LED1 켜기(ON):</b> 로봇 윗면의 LED 불빛을 켭니다.</li>
  <li style="margin-bottom: 5px;"><b>1000 밀리초(ms) 기다리기:</b> 1초(1000ms) 동안 켜진 상태를 그대로 유지합니다.</li>
  <li style="margin-bottom: 5px;"><b>내장 LED1 끄기(OFF):</b> 불빛을 다시 끕니다.</li>
  <li><b>1000 밀리초(ms) 기다리기:</b> 1초 동안 꺼진 상태를 유지합니다.</li>
</ol>

<div style="background: rgba(255, 255, 255, 0.05); padding: 12px; border-left: 4px solid #f39c12; border-radius: 4px; margin-top: 15px;">
  <b style="color: #f39c12;">💡 핵심 포인트</b><br>
  <div style="margin-top: 5px; color: #e0e0e0; line-height: 1.4;">2번~5번 과정이 <b>'계속 반복 실행(loop)'</b> 블록 안에 있기 때문에 전원을 끄기 전까지 로봇은 영원히 깜빡입니다! 기다리기(ms)의 숫자를 100이나 500으로 바꿔서 속도를 다르게 만들어 보세요!</div>
