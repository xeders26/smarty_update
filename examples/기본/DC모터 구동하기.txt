<h2 style="color: #3498db; margin: 0 0 5px 0;">⚙️ 기초 예제: DC 모터 앞뒤로 돌리기</h2>
<hr style="margin: 5px 0 15px 0; border-color: #555;">

<p style="margin: 0 0 15px 0; line-height: 1.5;">로봇의 바퀴를 움직이는 <b>DC 모터의 회전 방향과 속도를 제어하는 방법</b>을 배우는 기초 예제입니다.<br>
M1 모터가 1초는 앞으로, 1초는 뒤로 영원히 번갈아 가며 회전합니다.</p>

<h3 style="color: #3498db; margin: 0 0 10px 0;">🧩 블록 동작 흐름</h3>
<ol style="line-height: 1.5; margin: 0 0 15px 25px; padding: 0;">
  <li style="margin-bottom: 5px;"><b>스마티 시작하기 (setup):</b> 전원이 켜지면 모터를 움직이기 위한 기본 준비를 마칩니다.</li>
  <li style="margin-bottom: 5px;"><b>모터 M1 정회전:</b> 1번 DC 모터(M1)의 속도를 <b>300</b>으로 설정하여 작동시킵니다. (양수(+)는 정방향 회전)</li>
  <li style="margin-bottom: 5px;"><b>1000 밀리초(ms) 기다리기:</b> 모터가 300의 속도로 돌고 있는 상태를 1초 동안 <b>유지</b>합니다.</li>
  <li style="margin-bottom: 5px;"><b>모터 M1 역회전:</b> 1번 DC 모터(M1)의 속도를 <b>-300</b>으로 설정합니다. (음수(-)는 반대 방향 회전)</li>
  <li><b>1000 밀리초(ms) 기다리기:</b> 모터가 반대 방향으로 돌고 있는 상태를 다시 1초 동안 유지합니다.</li>
</ol>

<div style="background: rgba(255, 255, 255, 0.05); padding: 12px; border-left: 4px solid #f39c12; border-radius: 4px; margin-top: 15px;">
  <b style="color: #f39c12;">💡 핵심 포인트</b>
  <div style="margin-top: 5px; color: #e0e0e0; line-height: 1.5;">모터 블록은 한 번 명령을 내리면 다음 명령이 올 때까지 그 속도와 방향을 계속 유지합니다. 그래서 <b>반드시 모터 블록 뒤에는 시간(delay) 블록을 붙여주어야</b> 원하는 시간만큼 모터를 회전시킬 수 있습니다.<br><br>※ 속도를 0으로 설정하면 모터가 정지(Stop)합니다!</div>
